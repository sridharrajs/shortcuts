# Emacs

## File
c-x c-f 	open the file  
c-x c-s 	save the file  
c-x c-c 	close the file  

## Edit options
c-y		paste from clipboard  
m-w 	   	copies into clipboard  
c-w 	   	cut into clipboard  
c-space    	start marking area  
c-x u	   	undo  

## Selection  
c-x h 		select all  

## Navigation
c-x o 		switch to the other window  
c-s             search
